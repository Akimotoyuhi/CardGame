１.起動方法
CardGame.exeをダブルクリック
Webの場合は https://akimotoyuhi.github.io/CardGameRelease/ へ

２.操作方法
このゲームはマウスの左クリックのみで操作可能です。

３.概要
敵を倒しながらカードを集めてデッキを強化していくローグライクのカードゲームです。
ランダムに生成されるマップをいかに損害を抑えつつ進むか、どのカードをデッキに加えていくか、
という試行と選択が楽しいゲームとなっています。
現在２ステージまで遊べます。

４.ゲームの進め方
・タイトル画面
ゲームタイトルを「はじめる」と書かれた画像の上にドラッグ＆ドロップするとゲームが始まります。

・マップ画面
点滅しているマスがクリック可能のマスです。
一番右までたどり着ければゲームクリアです。
クリックしたマスによって異なる事が起こります。それぞれのマスの効果については下記に記載します。
戦闘→敵との戦闘
休憩→体力の回復とカードの強化が出来る
エリート→強い敵との戦闘。勝利するとレリックを獲得出来る
強敵→ボスとの戦闘　勝利するとステージクリア

各戦闘マスでは、勝利するとカードが報酬として獲得でき、その戦闘が難しいものほど報酬が豪華になります

・戦闘画面
左側がプレイヤー、右側が敵のキャラクターです。
全ての敵を倒すことが出来たら勝利、報酬を獲得できます。

画面下部のカードを対象にドラッグ＆ドロップする事で使用でき、使用する事でカードに書かれた効果が発動しますが
カードの左上に書かれたコストがプレイヤーの下部にあるコストより大きい場合は使用できません。

HPバーの下にアイコンがあったらマウスカーソルをその位置に持って行ってみましょう。効果を確認できます。
